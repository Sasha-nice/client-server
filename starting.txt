Запуск через командную строку
Сервер:
python3 server.py 

Клиент:
python3 client.py arg1 arg2 arg3 либо python3 client.py arg1 arg3

arg1 - задача
0 - Передать данные и тип задачи на сервер, отобразить идентификатор.
1 - Запросить и вывести статус задачи по идентификатору.
2 - Запросить и вывести результат выполнения задачи по идентификатору.
3 - Запуск в режиме, когда клиент отдает задачу на выполнение (данные и тип), дожидается ее выполнения, запрашивает и выводит результат выполнения задачи.

arg2 - функция обработки(присутствует при arg1 == 0 или arg1 == 3)

arg3 - данные
При arg1 == 0 или arg1 == 3 - текстовая строка
При arg1 == 1 или arg1 == 2 - идентификатор задачи.
